# Repositorio_atividade_gitFlow
**Atividade 3**
### Criar um novo repositorio 📦
- Garantir que ele já tenha main e develop
- Clonar o repositório
- Criar 3 funcionalidades(Features), diferentes.
- Sigam o passo a passo do gitFlow
- Após isso criar a realease
- Simular mudanças ou correções na release
- Enviar para o gitHub remoto e fazer merge com as branches necessárias
   
### Atenção: Não esqueça de sempre dar **Git pull**
